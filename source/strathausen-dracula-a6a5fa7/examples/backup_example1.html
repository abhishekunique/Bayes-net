Creating Bayesian Net:
  1. New Node to create node.
  2. Select to apply changes to node attributes and position.
  3. New Edge to connect nodes.
  4. Show Tables to enter CPT of node.
  5. Redraw to clean up the canvas.
D-Separation:
  1. Select D-Separation Nodes to choose nodes to d-separate. (First node is source node. Second node is target node. All subsequent nodes are observed nodes.)
  2. Check D-Separation.
